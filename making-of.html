Why We Chose These Colors for Our Website

Hey there! You might’ve noticed that our website uses a mix of beige, brown, and orange. These colors weren’t just picked randomly—they’re actually super important in setting the vibe we want you to feel.
Here’s why we chose them:

1. Beige and Brown: Cozy and Classic

These colors are all about warmth and comfort—kind of like how you feel when you step into a traditional Italian kitchen.
Brown reminds us of things like wood-fired ovens and fresh-baked bread, while beige adds a soft, inviting touch.
Together, they give our site a rustic, natural feel that matches the fresh, authentic ingredients we use.


2. Orange: Fun and Tasty

Orange brings the energy! It’s bright and happy, making our site feel lively and exciting—just like enjoying a plate of delicious pasta or pizza with friends. Fun fact: orange is known to make people feel hungrier (so maybe that’s why you’re thinking about what to order right now 😋).


3. How It All Comes Together

These colors work as a team. The beige and brown give you that cozy, Italian vibe, and the orange helps important things stand out, like our menu or special deals. Plus, this combo makes the website easy to use and fun to explore.

How HTML, CSS, and JS Affect How Easy It Is to Maintain Our Website

You probably don’t think too much about the code behind the website but it actually has a big impact on how easy it is for us.
Here’s how the three main types of code: HTML, CSS, JS help us make sure everything works well and stays easy to update:

1. HTML Structure of the Website

It's what makes the basic structure of the site, like headings, paragraphs, links, and pictures. Since it's easy to read and understand, we can quickly update or add new pages. We added comments to most of the lines to make the code easier to understand and search for. If we want to add something new, like a menu item or a new section, it’s simple to do because comments that we added make it clear where everything goes.


2. CSS (Design and Style)

CSS is all about how the website looks. We choose the colors, the font sizes, and layout. The cool thing is that, if we want to change something on the site (like making the buttons bigger or changing the colors), we don’t have to mess with the whole website. We can just update the CSS, and it changes the look everywhere at once. This makes it way easier for us to keep the site looking good without messing everything up. We also added some comments to make it way easier to understand.


3. JavaScript

JavaScript adds the interactive parts-like when buttons do something cool when you click them. JS makes the site feel more alive, but it also means we need to make sure it’s clean and well-organized so things don’t break. If the JS isn’t written well, it can mess up the site, so we make sure to keep it simple and clear so it’s easier to fix or update later.
